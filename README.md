----------
INSTALLATION
------------
Add the bundle to your composer.json
```
"doh/find-unused-bundles-bundle": "dev-master"
```
----------
Usage
------------
Scan your app
```
php app/console doh:unusedblundles:find
php app/console d:u:f
```
